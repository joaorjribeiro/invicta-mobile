# Invicta Mobile

App mobile de gestão financeira pessoal com autenticação, controle de transações e visualização de metas.

## Tecnologias

- React Native + Expo
- TypeScript
- Firebase (Auth + Firestore)

## Funcionalidades

- Autenticação (login e cadastro)
- Controle de transações (entrada e saída)
- Visualização de saldo e metas
- Dados sincronizados na nuvem

## Como rodar

```bash
npm install
npx expo start
```

## Stack

| Camada | Tecnologia |
|--------|-----------|
| Interface | React Native |
| Backend | Firebase |
| Banco de dados | Firestore |
| Tipagem | TypeScript |
